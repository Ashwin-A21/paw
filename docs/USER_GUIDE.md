# User Guide & Glossary - Paw Pal

## 1. Getting Started

### For Adoption

1. **Browse**: Go to the "Adopt" page to see animals looking for a home.
2. **Apply**: Click "Adopt Me" on a pet's profile. You must be logged in to apply.
3. **Wait**: Admins will review your application. Check your dashboard for status updates.

### For Reporting a Rescue

1. **Emergency**: If you see an animal in danger, click "Report Emergency".
2. **Details**: Provide a clear photo, location details, and urgency level.
3. **Submit**: Once submitted, local rescuers are notified through their dashboards.

### For Volunteers

1. **Tasks**: Check your dashboard for newly assigned tasks.
2. **Updates**: As you progress with a rescue, update the task status so admins can track progress.

## 2. Glossary of Terms

| Term                 | Definition                                                                                                             |
| -------------------- | ---------------------------------------------------------------------------------------------------------------------- |
| **Pet ID**           | A unique number assigned to every pet in the database for tracking.                                                    |
| **SOS Alert**        | A high-urgency rescue report that requires immediate attention.                                                        |
| **Verified Partner** | A breeder, shelter, or vet that has been vetted by the Paw Pal admin team.                                             |
| **GSAP**             | GreenSock Animation Platform, the library used for smooth visual transitions on the site.                              |
| **Tailwind CSS**     | A utility-first CSS framework used for most of the site's styling.                                                     |
| **RBAC**             | Role-Based Access Control, ensuring users only see what their role allows (e.g., volunteers don't see admin settings). |

## 3. Frequently Asked Questions (FAQ)

**Q: Is there a fee for adoption?**  
A: Paw Pal itself is free. However, specific shelters or breeders may have their own processing fees.

**Q: How do I become a verified rescuer?**  
A: Register for an account and contact the admin team with your credentials to have your role updated.
